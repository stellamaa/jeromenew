# 00ff00.systems

A React application recreating the minimalist artist portfolio website for Sandra Golubjevaite.

## Features

- Interactive radio button navigation
- Clean, minimalist design with white background and black text
- Green accent colors for links and decorative elements
- Responsive layout
- Large decorative letters at the bottom
- Show/hide content functionality based on radio button selection

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project Structure

- `src/App.js` - Main application component with state management
- `src/App.css` - Styling for the application
- `src/index.js` - React entry point
- `public/index.html` - HTML template

## Functionality

The radio buttons control which content section is displayed:
- "about" - Shows the artist bio and contact information
- "upcoming" - Shows upcoming exhibitions and events

The design features a grid of empty squares and large decorative letters spelling "onegu" at the bottom of the page.
